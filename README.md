# LbTab.js插件
## 功能：滑动门切换，可自定义html中的结构和切换事件类型。
## 引入文件：
使用gulp构建，具体实例中需要引入路径build\web\js下的lbTab.js或lbTab.min.js；
css样式可以自定义也可使用路径build\web\css下的lbTab.css或lbTab.min.css；
## 使用说明：
new LbTab(id, event, active);
传入外容器id；
相应事件类型（可选，默认为click）；
切换样式名称（可选，默认为active）；
